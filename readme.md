# 集市 主题


## 如何安装

### 1 新建 be 项目

    composer create-project be/new

### 2 安装主题（be/theme-market）

    composer require be/theme-market



## 部件

1. 头部工具栏 - HeaderToolbar
2. 头部 - Header
3. 轮播图 - Slider
4. 页面主体标题 - PageTitle
5. 横幅 - Banner
6. 多个横幅 - Banners
7. 底部 - Footer
